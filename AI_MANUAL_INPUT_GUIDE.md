# 🤖 AI dengan Manual Input - Panduan

## Overview

Sekarang Anda bisa menggunakan AI untuk generate pesan TANPA perlu upload CSV! Cukup input nomor telepon manual di textarea, aktifkan AI, dan biarkan AI generate pesan unik untuk setiap nomor.

---

## Cara Menggunakan

### Metode 1: Dengan CSV (Lebih Personal)

1. Upload CSV dengan data lengkap (phone, name, company, dll)
2. Enable "Gunakan AI untuk Generate Pesan Personal"
3. Tulis template dengan placeholder: `{name}`, `{company}`, dll
4. AI akan generate pesan berdasarkan data CSV

**Contoh Template:**
```
Buatkan pesan untuk {name} dari {company}.
Tawarkan layanan IT dengan diskon 20%.
```

**Hasil:**
```
Halo Pak John Doe dari PT ABC Technology! 👋
Kami ingin menawarkan diskon spesial 20%...
```

---

### Metode 2: Manual Input (Tanpa CSV) ⭐ NEW!

1. Input nomor telepon di textarea (satu per baris)
   ```
   628123456789
   628987654321
   628555666777
   ```

2. Enable "Gunakan AI untuk Generate Pesan Personal"

3. Tulis instruksi untuk AI (TANPA placeholder)
   ```
   Buatkan pesan promosi tentang produk IT kami.
   Tawarkan diskon 20% dan konsultasi gratis.
   Gunakan bahasa profesional tapi ramah.
   Ajak untuk meeting.
   ```

4. Click "🤖 Generate Pesan AI"

5. AI akan generate pesan UNIK untuk setiap nomor!

**Hasil:**
```
Nomor 1:
Halo! 👋 Kami dari PT Digital Solutions ingin menawarkan...

Nomor 2:
Selamat pagi! Perkenalkan kami dari PT Digital Solutions...

Nomor 3:
Halo, salam kenal! Kami perusahaan IT yang fokus pada...
```

Setiap pesan BERBEDA dan UNIK! 🎉

## Tips & Best Practices

### ✅ DO:

1. **Instruksi yang Jelas**
   ```
   ❌ Bad: "Buatkan pesan bagus"
   ✅ Good: "Buatkan pesan promosi produk X dengan benefit Y, ajak untuk Z"
   ```

2. **Spesifik tentang Tone**
   ```
   - "Profesional tapi ramah"
   - "Casual dan friendly"
   - "Formal dan sopan"
   - "Enthusiastic dan energetic"
   ```

3. **Berikan Konteks**
   ```
   Tambahkan di "Konteks Tambahan":
   "Kami perusahaan IT yang fokus pada automation.
   Target: SME di Indonesia.
   USP: Affordable dan easy to use."
   ```

4. **Test dengan 2-3 Nomor Dulu**
   - Generate untuk 2-3 nomor
   - Review hasilnya
   - Adjust instruksi jika perlu
   - Baru kirim ke semua

### ❌ DON'T:

1. **Instruksi Terlalu Umum**
   ```
   ❌ "Buatkan pesan"
   ❌ "Kirim promosi"
   ```

2. **Terlalu Panjang**
   ```
   ❌ Instruksi 10 paragraf
   ✅ 3-5 kalimat yang jelas
   ```

3. **Tidak Review**
   ```
   ❌ Generate langsung kirim
   ✅ Generate → Review → Edit jika perlu → Kirim
   ```

## FAQ

**Q: Apakah setiap pesan benar-benar unik?**
A: Ya! AI generate pesan berbeda untuk setiap nomor.

**Q: Berapa lama generate 10 nomor?**
A: Sekitar 10-30 detik tergantung kompleksitas.

**Q: Apakah bisa pakai bahasa Indonesia?**
A: Ya! AI support multi-language.

**Q: Apakah gratis?**
A: Free tier Gemini: 1,500 requests/day. Cukup untuk 1,500 pesan/hari.

**Q: Bagaimana jika AI generate pesan yang tidak sesuai?**
A: Review dulu sebelum kirim. Bisa edit instruksi dan generate ulang.

**Q: Apakah bisa combine CSV + Manual?**
A: Tidak. Pilih salah satu: CSV atau Manual input.

**Q: Lebih baik CSV atau Manual?**
A: 
- CSV: Jika punya data lengkap dan mau lebih personal
- Manual: Jika cepat dan tidak punya data detail

## Troubleshooting

**Issue: AI generate pesan yang sama**
- Solution: Buat instruksi lebih spesifik dan variatif

**Issue: Pesan terlalu panjang**
- Solution: Tambahkan "Max 2-3 paragraf" di instruksi

**Issue: Pesan tidak sesuai tone**
- Solution: Spesifik tentang tone: "profesional", "casual", "formal"

**Issue: Generate lambat**
- Solution: Normal untuk 10+ nomor. Tunggu 30-60 detik.
